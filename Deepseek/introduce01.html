CSS链接标签解析

理解<link rel="stylesheet" href="./site.css">的含义与应用

代码含义解析

这行HTML代码用于将外部CSS样式表链接到当前HTML文档：

<link rel="stylesheet" href="./site.css">

组成部分解析：

<link> - HTML标签，用于链接外部资源到当前文档

rel="stylesheet" - 定义链接资源与当前文档的关系，此处表示链接的是样式表

href="./site.css" - 指定外部CSS文件的路径：

  • ./ - 表示当前目录（与HTML文件相同的目录）
  • site.css - CSS文件的名称

注意： 这行代码应该放置在HTML文档的<head>部分，以确保样式在页面加载时正确应用。

文件结构示例：

项目目录/
  index.html
  site.css
  images/
    logo.png

实际应用演示

下面的元素样式是通过类似的外部CSS文件应用的：

主色调元素

这个元素的样式是通过CSS类".primary"定义的

次要色调元素

这个元素的样式是通过CSS类".secondary"定义的

成功状态元素

这个元素的样式是通过CSS类".success"定义的

site.css 文件内容示例：

/* 基础样式 */
body {
  font-family: Arial, sans-serif;
  line-height: 1.6;
  margin: 0;
  padding: 0;
  background-color: #f4f4f4;
}

/* 主色调 */
.primary {
  background-color: #6a11cb;
  color: white;
  padding: 15px;
  border-radius: 5px;
}

/* 次要色调 */
.secondary {
  background-color: #2575fc;
  color: white;
  padding: 15px;
  border-radius: 5px;
}

/* 成功状态 */
.success {
  background-color: #2ecc71;
  color: white;
  padding: 15px;
  border-radius: 5px;
}

使用外部CSS的优势

  • 分离内容与表现 - HTML负责结构，CSS负责样式
  • 易于维护 - 修改网站样式只需更改CSS文件，无需修改每个HTML页面
  • 提高效率 - 一个CSS文件可以被多个页面使用
  • 缓存优势 - 浏览器会缓存CSS文件，加快页面加载速度
  • 一致性 - 确保整个网站的风格统一
